# Hajira Malik - Portfolio

This is a simple, premium portfolio website built with HTML, CSS, and JavaScript.

## How to View

1.  Simply double-click the `index.html` file to open it in your web browser.
2.  Alternatively, you can drag and drop `index.html` into a browser tab.

## Structure

-   `index.html`: The main content and structure of the website.
-   `style.css`: All the styling, fonts, and layout rules.
-   `script.js`: Adds smooth interactions like the custom cursor and scroll animations.

## Customization

-   **Images**: Replace the gray placeholders in the `index.html` with your actual images. Look for `<div class="placeholder-img">` or `<div class="placeholder-project">` and replace them with `<img>` tags.
-   **Colors**: You can change the colors in `style.css` under the `:root` section.
